Welcome to The AFP Project

The AFP Project is a set of tools to easly bridge user dekstops, data-center servers and the public-cloud.

It consists of the following components:

  • afp-core
  • afp-web
  • afp-cli
  • afp-alppaca

For further details and an introduction please see this talk from the Chemnitzer Linux Tage 2016.
